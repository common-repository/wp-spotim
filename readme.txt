WP Spot.IM let you add and manage your chats from Spot.IM.

== Description ==

Add chat to your wordpress site. 

Easy to use just grab the code to your chat from Spot.IM and paste him in the WP Spot.IM page in the admin, and it will be shown on your website.

== Installation ==

1. Upload the plugin to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to "WP Spot.IM" page under Settings in the admin menu
4. Grab your code from http://Spot.IM and paste him in the code field
5. Click Update and your chat is now on your website
